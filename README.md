## Panel de administración de estacionamientos

### Frontend NextJS

Para ejecutar el panel en modo de desarrollo:

``` npm run dev ```

### Backend ExpressJS

Instalar dependencias:

``` npm install ``` || ``` pnpm install ```

Iniciar el servidor de desarrollo:

``` npm run dev ``` || ``` pnpm run dev ```
